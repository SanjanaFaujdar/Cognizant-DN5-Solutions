# Git HOL 5 - Clean Up and Push Back to Remote Git

## Objectives
- Explain how to clean up and push back to remote Git

## Steps Performed

```bash
# 1. Verify master is in clean state
git status

# 2. List all available branches
git branch -a

# 3. Pull the remote git repository to master
git pull origin master

# 4. Push pending changes from Git-HOL-4
git push origin master

# 5. Observe changes reflected in remote repository
# Open GitHub/GitLab in browser and verify
```

## Expected Output
- Master branch is in clean state
- All branches listed successfully
- Remote repository pulled successfully
- All pending changes pushed to remote
- Changes visible in GitHub/GitLab repository
